# Environment

## Beginner Setup

Leave these unchanged:

```env
SLSKD_DOWNLOADS_PATH=./downloads
FINAL_MUSIC_PATH=./music
```

Only change them if:
- slskd downloads somewhere else
- Jellyfin uses another folder
- you run Soulkeep on a server

Required:

```env
SLSKD_API_URL=http://localhost:5030/api/v0
SLSKD_API_KEY=YOUR_SLSKD_API_KEY
```
